1. EXECUTIVE SUMMARY
================================================================================

Maxera Talent is a United States-based contingent recruitment and staffing firm.
This document defines all product, functional, and technical requirements for the
design, development, and launch of the Maxera Talent website.

The website serves a dual audience:
  (1) Companies seeking to hire qualified professionals
  (2) Job seekers looking for career opportunities

The platform must establish credibility, generate qualified leads from hiring
managers, and attract strong candidates across key industries.

Project Info:
  Business Model  : Contingent Recruitment / Staffing Firm
  Primary Goal    : Lead generation from employers + candidate pipeline
  Secondary Goal  : Brand authority in the US staffing market
  Target Market   : United States – nationwide
  Launch Scope    : 8 pages, fully responsive, Vercel-deployed


================================================================================
2. PROJECT OBJECTIVES
================================================================================

BUSINESS GOALS
- Convert website visitors (employers) into active hiring clients via lead forms
- Build a qualified candidate pipeline through job listings and resume submission
- Establish Maxera Talent as a credible staffing brand in the US market
- Support SEO growth to capture organic search traffic from target personas

PRODUCT GOALS
- Deliver a fast-loading, mobile-first website meeting Core Web Vitals standards
- Implement accessible, component-based code architecture for scalability
- Provide seamless UX across all 8 pages with clear conversion paths
- Enable form-based data capture for employers and candidates

5. DETAILED PAGE REQUIREMENTS
================================================================================

--------------------------------------------------------------------------------
5.1 HOME PAGE (/)
--------------------------------------------------------------------------------

PURPOSE:
Primary landing page. Establish brand identity, communicate value proposition,
drive employer and candidate conversions.

SECTIONS REQUIRED:

[1] Hero Section
    - Full-width hero with headline, sub-headline
    - Two CTAs: "Hire Talent" (primary) and "Find Jobs" (secondary)
    - Background: dark navy with geometric or abstract overlay
    - Trust indicators: serving US companies, placements count, etc.

[2] Company Introduction
    - 2–3 sentence overview of Maxera Talent's mission and specialisation
    - Supporting stat or credibility badge (optional MVP)

[3] Services Overview
    - Three service cards: Contingent Recruitment, Contract Staffing, Direct Hire
    - Icon, title, 1-line description, and "Learn More" link per card

[4] Industries Section
    - 5 industry tiles: Technology, Healthcare, Finance, Engineering, Business Ops
    - Icon + label layout, links to Industries page

[5] Hiring Process Section
    - 5-step numbered flow:
      1. Understand Requirements
      2. Source Candidates
      3. Screen Candidates
      4. Coordinate Interviews
      5. Successful Placement

[6] Testimonials Section (optional for MVP)
    - 2–3 placeholder quotes with name, title, company

[7] Call to Action Banner
    - Headline: "Ready to Build Your Team?"
    - CTA Button: "Hire Talent Today" — links to /employers


--------------------------------------------------------------------------------
5.2 ABOUT PAGE (/about)
--------------------------------------------------------------------------------

PURPOSE:
Build trust and credibility. Communicate company story, values, and the people
behind Maxera Talent.

SECTIONS REQUIRED:
- Page Hero: heading + short description
- Company Overview: 2–4 paragraphs on background and approach
- Mission Statement: "Our Mission is to…" — prominent display
- Vision Statement: "Our Vision is to…"
- Core Values: 4–6 values as cards or grid
  Examples: Integrity, Speed, Partnership, Excellence, Transparency, Quality
- Team Section (optional for MVP): placeholder cards with role titles


--------------------------------------------------------------------------------
5.3 SERVICES PAGE (/services)
--------------------------------------------------------------------------------

PURPOSE:
Explain each service offering in detail to help employers understand what
Maxera Talent delivers.

SERVICE 1 — CONTINGENT RECRUITMENT
  Definition : Placement fee paid only upon successful hire. No upfront cost.
  Ideal for  : Companies looking to reduce hiring risk
  Include    : Key benefits, typical timeline, process overview

SERVICE 2 — CONTRACT STAFFING
  Definition : Temporary professionals placed for a defined project or period
  Ideal for  : Project-based needs, seasonal demand, backfill situations
  Include    : Key benefits, typical engagement duration

SERVICE 3 — DIRECT HIRE / PERMANENT PLACEMENT
  Definition : Full-time placement with guaranteed replacement period
  Ideal for  : Building long-term teams with committed professionals
  Include    : Key benefits, guarantee terms

5.6 CANDIDATES PAGE (/candidates)
--------------------------------------------------------------------------------

PURPOSE:
Attract and convert job seekers. Give them a clear reason to share information.

SECTIONS REQUIRED:

================================================================================
8. TECHNICAL REQUIREMENTS
================================================================================

TECH STACK:
  Framework   : Next.js 14 (App Router)
  Language    : TypeScript (strict mode)
  Styling     : Tailwind CSS (utility-first, no CSS-in-JS)
  Forms       : React Hook Form + Zod
  Hosting     : Vercel (CI/CD via GitHub)
  SEO         : Next.js Metadata API (per-page metadata, OpenGraph, sitemap.xml)
  Images      : Next.js Image component (lazy load, WebP optimisation)
  Icons       : Lucide React or HeroIcons

PERFORMANCE TARGETS:
  Lighthouse Performance (mobile)  : >= 90
  Largest Contentful Paint (LCP)   : < 2.5s
  First Input Delay (FID)          : < 100ms
  Cumulative Layout Shift (CLS)    : < 0.1
  Page bundle size (gzipped)       : < 200kB per page

SEO REQUIREMENTS:
  - Unique <title> and meta description per page
  - Semantic HTML: correct h1–h3 hierarchy
  - Open Graph and Twitter Card tags
  - robots.txt and sitemap.xml auto-generated
  - Structured data (JSON-LD) on Jobs page for Google Jobs indexing

12. DEFINITION OF DONE / ACCEPTANCE CRITERIA
================================================================================

- All 8 pages live, linked correctly, no errors on Chrome/Firefox/Safari/Edge
- All 3 forms validate correctly and show success states on submission
- Resume upload accepts PDF/DOC up to 5MB; rejects invalid types with error
- Website fully usable on mobile (320px+), tablet (768px+), desktop (1280px+)
- Lighthouse scores >= 90 for Performance, Accessibility, Best Practices, SEO
- Navigation consistent across all pages with active link state
- All CTAs link to the correct destination pages
- No broken links, missing images, or console errors in production
- Site deployed to Vercel with custom domain or Vercel subdomain
- robots.txt and sitemap.xml accessible at root
